BnBnで得られるメリット

BnBnでは、チラシ・ポスターデザインの依頼をクリエイターを志望する学生へ安価で発注することができます。
希望する企業様は参加希望もしくはお問い合わせより、意思をご表明ください。

登録の手順

下記画像のように、まずは説明会に参加しましょう。提出資料は以下のリンクより雛形をダウンロードし、適宜使用してください。

応募

登録までの流れ

登録が完了すると、学生に広告を依頼できます。

  1. ページ下部のダウンロードリンクより、「登録申請用紙」をダウンロードして必要事項を記入します。
  2. 登録申請用紙をBnBnに提出します
  3. 大学サイドで審査
  4. 通過通知で登録完了

＊登録が完了したご依頼主様には学生に公募するにあたって、また、学生と打ち合わせをしていく過程の中での諸注意などをオリエンテーションでご説明します（出席必須・代理出席可）

学生に広告デザインを依頼するまでの流れ

  1. 登録を完了する
  2. 「応募書類ダウンロード」から「依頼申請用紙」をBnBnに提出
  3. 「募集案件」に情報がアップされ、学生に募集内容が通知される
  4. 学生から応募
  5. 代表者に学生の連絡先をお知らせし、マッチング完了
  • 「依頼申請用紙」に入力頂きました内容が学生に発信されます。
  • 公募した場合でも学生から応募がこない場合もあります。ご了承ください。
    （申込締切までの製作期間の問題や試験や課題、行事など）
  • 応募があった学生にはご依頼主様から学生のメールアドレスにご連絡を入れてください。
  • ご依頼製作物ができましたらBnBnにも送付ください。
  • ご依頼主様は、ご希望に合った広告デザインができたかどうかなどのフィードバックにご協力ください。
